# CT_Assessment

A basic Flutter shopping list for Chunky Tofu Studios.

## Set Up
- Clone the repo
- Make sure you have all the dependencies with `flutter pub get`
- Run the app using whatever method you prefer (`flutter run`, VS Code's Run and Debug button, etc)

### State Management

I used BLoC for state management because it was the default and prefered option.

I spent around four hours working on this project.
